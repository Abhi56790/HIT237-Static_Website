  • Machine learning approaches for Cyber Security

    This project explores various machine learning techniques for enhancing cyber security measures, including anomaly detection, intrusion detection, and malware classification.

    Supervisor: Bharanidharan Shanmugam

  • Informetrics applications in multidisciplinary domain

    This project investigates the use of informetrics in diverse fields, such as healthcare, education, and social sciences, to analyze information flow, usage patterns, and citation networks.

    Supervisor: Yakub Sebastian

  • Development of a Virtual Reality System to Test Binaural Hearing

    This project aims to develop a virtual reality system for assessing binaural hearing abilities, providing a realistic environment for evaluating auditory perception and spatial localization.

    Supervisor: Sami Azam

  • Current trends on cryptomining and its potential impact on cryptocurrencies

    This project analyzes the latest trends in cryptomining activities, exploring their implications for cryptocurrencies, including security risks, market dynamics, and regulatory challenges.

    Supervisor: Sami Azam

  • Artificial Intelligence in Health Informatics

    This project investigates the application of artificial intelligence techniques, such as machine learning and natural language processing, in health informatics for tasks like disease diagnosis, patient monitoring, and healthcare data analysis.

    Supervisor: Asif Karim

  • Unsupervised Model Development from Autism Screening Data

    This project aims to develop unsupervised machine learning models using autism screening data to identify patterns and subgroups within the autistic population, facilitating personalized interventions and treatment strategies.

    Supervisor: Asif Karim

  • Applying Artificial Intelligence to solve real world problems

    This project explores the practical applications of artificial intelligence algorithms and techniques to address real-world challenges across various domains, such as healthcare, finance, transportation, and environmental sustainability.

    Supervisor: Bharanidharan Shanmugam
